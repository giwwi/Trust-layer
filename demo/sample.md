# Review Capacity as the Bottleneck

AI systems have made it much cheaper to produce analytical drafts, policy memos, white papers, and proposal fragments. This increases the number of texts that look plausible enough to request human attention.

The constraint has not moved at the same speed. Expert review remains slow, contextual, and scarce. A useful system should therefore help route attention, not pretend to settle truth.

The main proposal is a bounded first-pass triage layer. It should identify the probable central argument, the main claim zones, the areas where evidence is doing work, and the places where uncertainty remains important.

The weak point is that fast triage can become bureaucratic if every text is forced into a polished artifact. The review process should stop early when a document does not justify more attention, and escalate only when the next decision would change.
