# Public Release Checklist

Use this checklist before sharing Trust Layer publicly.

- [ ] No API keys in frontend code.
- [ ] No API keys committed to GitHub.
- [ ] Example mode works without backend.
- [ ] Live AI mode has character limit.
- [ ] Live AI mode has rate limit.
- [ ] Live AI mode has friendly fallback.
- [ ] Raw localhost errors are hidden.
- [ ] Privacy note is visible.
- [ ] “Not a truth engine / not AI detector / not expert review” is visible.
- [ ] English version works.
- [ ] Russian version does not block usage.
- [ ] Mobile layout is usable.
- [ ] README explains public demo modes.
- [ ] Show HN post text prepared.
- [ ] Public URL tested in incognito browser.
